# Greenify — Lifestyle Coach for Sustainable Habits

Greenify helps users adopt lower‑carbon habits, tracks estimated environmental impact, and connects them with local resources and challenges.

Goals
- Provide personalized micro‑habits with transparent CO2e estimates
- Track user impact (kg CO2e, money, water)
- Offer local resources and gamified challenges

Tech stack (suggested)
- Mobile: React Native (TypeScript)
- Backend: Node.js + TypeScript (Fastify or Express) or Python (FastAPI)
- Database: PostgreSQL
- CI: GitHub Actions

Getting started (local)
1. Clone repository
2. Install backend deps: `cd backend && npm install`
3. Install mobile deps: `cd mobile && npm install`
4. Run locally (see each subfolder README for details)
